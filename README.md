## 🚀 Currently Building

### MLForge
A full-stack web platform for uploading datasets, training and comparing ML models (Logistic Regression, Decision Tree, Random Forest, KNN), and making predictions.

**Core Features:**
- 🔐 JWT-based authentication
- 📊 CSV dataset upload with automatic stat generation
- ⚙️ One-click training across 4 ML algorithms
- 📈 Visual model comparison dashboard
- 🎯 Live prediction interface

**Tech Stack:** FastAPI · SQLAlchemy · scikit-learn · pandas · HTML/CSS/JavaScript
